# Google Sites Chat

Simple static chat website for GitHub Pages and Google Sites embedding.

## Publish

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select `main` and `/ (root)`.
6. Save.

GitHub Pages will publish a URL like:

- `https://prithivp12.github.io/Unblocked_Chat.github.io/`

## Embed in Google Sites

1. Open your Google Site.
2. Click `Insert` > `Embed`.
3. Paste the GitHub Pages URL.
4. Resize the frame and publish.

## Note

The page asks for the Gemini API key in the browser instead of storing it in the repo.
